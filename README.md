# 🤖 AI Chatbot Agent (FastAPI + LLMs)

An intelligent AI chatbot built using Large Language Models (LLMs), designed to perform tasks like answering queries, web search, and reasoning using external tools.

---

## 🚀 Features

- 🔹 LLM-powered chatbot (OpenAI / Groq)
- 🔹 Tool usage (Tavily Search API)
- 🔹 FastAPI backend
- 🔹 Agent-based reasoning (ReAct / Tool calling)
- 🔹 Scalable and modular architecture

---

## 🛠️ Tech Stack

- Python
- FastAPI
- LangChain / Agent Framework
- OpenAI / Groq APIs
- Tavily Search API
